# DPAK-s-WebOS\


This is my very own WebOS, 
It is steph Curry themed
it uses HTML, CSS, and JavaCript

# how to use
The project is open source, anyone can download the code
and run it locally or you can check out the deployed site


#Features 
Draggable Windows: Click and drag the title bars to move the content windows anywhere on the screen.
Live task bar: Features a real-time clock and a simulated, dynamically updating battery percentage.
semi transparent windows: Semi-transparent windows with a clean `backdrop-filter: blur()` effect for a modern look.
